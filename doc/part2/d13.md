## LED 불켜기

```
sudo apt-get update
sudo apt-get install python-rpi.gpio

sudo python
>>>import RPi.GPIO as GPIO
>>>GPIO.setmode(GPIO.BCM)
>>>GPIO.setup(25, GPIO.OUT)
>>>GPIO.setup(25, GPIO.HIGH)
>>>GPIO.setup(25, GPIO.LOW)

```
