## LED 불켜기

![실물](/doc/img/rasledr.jpg)

```
sudo apt-get update
sudo apt-get install python-rpi.gpio

sudo python
>>>import RPi.GPIO as GPIO
>>>GPIO.setmode(GPIO.BCM)
>>>GPIO.setup(17, GPIO.OUT)
>>>GPIO.setup(17, GPIO.HIGH)
>>>GPIO.setup(17, GPIO.LOW)

```
